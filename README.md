# cifar10
<p>一个cifar10，可以识别10种物体</p>
<p>环境是pytorch2.8.0,python3.9,cuda12.9</p>
<p>model文件夹存放了模型的代码</p>
<p>data文件夹存放了具体的模型</p>
<p>varients文件夹存放了实验用的一些模型</p>
<p>experiment_plot文件夹存放了实验数据和实验用的模型，部分实验数据的生成是在cifar10.ipynb,experiment.ipynb,datavisualize.ipynb里面改改参数弄出来的，没有完全展示在文件中</p>
<p>result文件夹存放了成果</p>
<p>data.txt存放了原始数据</p>
<p>report.txt存放了生成报告的代码</p>
<p>report.pdf存放了报告</p>
